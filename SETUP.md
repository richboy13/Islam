# Инструкция по настройке GitHub Pages

## Вариант 1: Через настройки репозитория (самый простой)

1. Перейдите в настройки репозитория:
   https://github.com/richboy13/Islam/settings/pages

2. В разделе "Source" выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)` или `/docs`
   - Нажмите **Save**

3. Подождите 1-2 минуты, сайт будет доступен по адресу:
   https://richboy13.github.io/Islam/

## Вариант 2: Через GitHub Actions (уже настроено)

1. Перейдите в раздел Actions:
   https://github.com/richboy13/Islam/actions

2. Если workflow не запустился автоматически, нажмите "Run workflow"

3. После успешного выполнения workflow сайт будет доступен

## Проверка статуса

- Проверьте статус деплоя в разделе Actions
- Убедитесь, что в настройках Pages выбран правильный источник
- Проверьте, что файл `index.html` находится в корне или в папке `docs`

## Если сайт не работает

1. Убедитесь, что репозиторий публичный (не private)
2. Проверьте, что файл `index.html` существует
3. Подождите несколько минут после настройки
4. Очистите кэш браузера и попробуйте снова
